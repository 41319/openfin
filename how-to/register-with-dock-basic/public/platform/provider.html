Basic Dock Example

Demonstrate basic dock interactions.

Please click the register button to add a Dock Provider (it will start up Dock and OpenFin Workspace if it isn't already running). Once a dock is registered the show/hide buttons will be enabled. Use show to show the dock you just created. Deregister will remove your dock entry from OpenFin Workspace.

This is a simple example showing how you can hand code a Dock.
